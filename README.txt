# FreshMarket – E-Commerce Web Application
FreshMarket is a university final project built with Java 21, Jakarta EE (JSF), and MySQL.
It is a simplified e-commerce platform that demonstrates backend and fullstack development skills.

# Features
- User registration and authentication (login/logout)  
- Product catalog with CRUD operations (create, update, delete)  
- Shopping cart and checkout flow  
- Admin dashboard for managing users and products  
- Database integration with MySQL  

# Technologies
- Java 21 (JDK 21) – backend logic
- Jakarta EE (JSF) – server-side framework
- MySQL – database
- JDBC – database connectivity
- Maven – project management and build tool
- HTML/CSS/JavaScript – basic frontend

# Requirements
- JDK 21
- MySQL server
- Apache Tomcat 10.1
